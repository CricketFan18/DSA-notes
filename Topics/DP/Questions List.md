### 1. 1D DP (The Fundamentals & Variations)

You know Climbing Stairs, but these add twists like circular arrays or dictionary lookups.

- **House Robber** (LeetCode 198) - _Classic "no two adjacent elements" problem._
    
- **House Robber II** (LeetCode 213) - _The circular array twist._
    
- **Decode Ways** (LeetCode 91) - _Fibonacci logic, but with string parsing rules._
    
- **Word Break** (LeetCode 139) - _1D DP mixed with a Hash Set._
    

### 2. DP on Grids (Pathfinding)

You evaluated a grid scenario earlier. Now it is time to code them.

- **Unique Paths** (LeetCode 62) - _The absolute baseline grid problem._
    
- **Minimum Path Sum** (LeetCode 64) - _Grid DP with costs._
    
- **Dungeon Game** (LeetCode 174) - _A brilliant twist where you have to build your DP table from the bottom-right to the top-left._
    

### 3. Knapsack Variations (The "Fake Mustaches")

You have the framework for these; now it is about recognizing them in the wild.

- **Partition Equal Subset Sum** (LeetCode 416) - _Secretly just 0/1 Knapsack._
    
- **Target Sum** (LeetCode 494) - _0/1 Knapsack with positive and negative signs._
    
- **Coin Change II** (LeetCode 518) - _Unbounded Knapsack, but asking for "total combinations" instead of "minimum coins."_
    

### 4. String & 2D DP (LCS Family)

These build directly on the LCS logic you just perfectly derived.

- **Longest Palindromic Subsequence** (LeetCode 516) - _The 2-line trick question I mentioned earlier._
    
- **Edit Distance** (LeetCode 72) - _The absolute holy grail of String DP. If you can do this, you can do any string problem._
    
- **Distinct Subsequences** (LeetCode 115) - _String matching where you count the number of valid paths._
    

### 5. State Machine DP (The FAANG Favorites)

This is where we introduce a "status" variable to your state (e.g., "Am I holding a stock?").

- **Best Time to Buy and Sell Stock with Cooldown** (LeetCode 309) - _The perfect introduction to State Machines._
    
- **Best Time to Buy and Sell Stock with Transaction Fee** (LeetCode 714) - _A slight variation to test your state definitions._
    

### 6. Interval DP / Partition DP (The Advanced Tier)

Instead of moving left-to-right, you are looking at a range `[left, right]` and guessing where to make a "cut" in the middle.

- **Palindrome Partitioning II** (LeetCode 132) - _Where to cut a string so every piece is a palindrome._
    
- **Burst Balloons** (LeetCode 312) - _The final boss of Interval DP._
